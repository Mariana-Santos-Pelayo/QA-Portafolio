# Urban Routes – Test Cases

## Descripción del proyecto
Este proyecto consiste en pruebas manuales realizadas sobre la aplicación Urban Routes. El objetivo fue validar funcionalidades principales, detectar errores y documentar resultados.

## Alcance de pruebas
- Registro de usuario
- Inicio de sesión
- Solicitud de viajes
- Validación de formularios
- Manejo de errores

## Técnicas utilizadas
- Diseño de casos de prueba
- Pruebas funcionales
- Pruebas de regresión
- Reporte de bugs

## Herramientas
- Jira
- DevTools
- Google Sheets / Excel
